Stepper

Flat Styled jQuery stepper

View project on GitHub

What is stepper?

Stepper is an flat styled jquery number input field with stepper buttons.

Installation 

                        

                            <link
                            rel=
                            "stylesheet"
                            href=
                            "css/stepper.min.css"
                            />
                        

                        

                            <script
                            src=
                            "js/stepper.min.js"
                            ></script
                            >

Usage 

                    
                        $('.stepper').stepper();
